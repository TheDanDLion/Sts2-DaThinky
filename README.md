# Sts2-DaThinky
A Slay the Spire 2 mod that adds a calculator to combat that can be toggled on and off using the button on the top bar, or by pressing (C).

Known Issues:
* There's no sound when pressing the calcuator button.
* There's no tooltip. I haven't figured that one out yet.
* Spamming the button rapidly causes the animation to get out of sync
